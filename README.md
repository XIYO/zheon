# Zheon

YouTube 영상의 자막과 댓글을 수집·분석해 요약과 인사이트를 제공하는 SvelteKit + Supabase 애플리케이션.

---

기능

- 자막/댓글 수집 및 저장
- AI 요약 (Gemini)
- 카테고리/태그 자동 생성
- 콘텐츠 메트릭(교육성/오락성/정확성/명확성/깊이)
- 커뮤니티 감정·연령대 분석
- 실시간 업데이트 + 무한 스크롤
- 반응형 UI

---

투두

- i18n 도입 (보류)
- TTS: AI SDK 호환 음성 → Gemini → Google TTS → OpenAI 순
- 자막 번역 + TTS 오버레이
- 문서 정비 및 예제 강화
